1) create_phmm_input includes scripts for creating input for PhyloNet-HMM analysis. 

2) output_visualizations includes R scripts for visualizing the output of the PhyloNet-HMM analysis. 

3) process_hmm_output includes scripts for processing the raw output of PhyloNet-HMM.

4) run_phmm includes the scripts for running PhyloNet-HMM on an HPC.
