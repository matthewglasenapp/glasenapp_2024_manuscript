These files are used to visualize the posterior probabilities of introgression for the arachidonate-5-lipoxygenase gene and surrounding area. These files are used as input to the script scripts/phylonet_hmm/output_visualizations/arachidonate-5-lipoxygenase/arachidonate_introgressioin_tract.R

1) arach.gff contains the gff3 record for the arachidonate-5-lipoxygenase gene.

2) glycosyltransferase.gff contains the gff3 record for the glycosyltransferase gene. 

3) lysosomal.gff contains the gff3 record for the lysosomal gene. 

4) NW_022145601.1.json contains the PhyloNet-HMM posterior decoding probabilities for the scaffold NW_022145601.1. 

5) NW_022145601_1_coordinates.txt contains the coordinates corresponding the posterior decoding probabilities for the scaffold NW_022145601.1.
