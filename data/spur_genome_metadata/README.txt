1) GeneGoTerms.txt contains the gene ontology terms from echinobase.org for the S. purpuratus reference genome. 

2) GeneKoTerms.txt contains the KEGG ontology terms from echinobase.org for the S. purpuratus reference genome. 

3) GenePageGeneralInfo_AllGenes.txt contains metadata on annotated genes in the S. purpuratus reference genome. 

4) nonoverlapping_unique_CDS.bed contains all nonoverlapping, unique CDS sequence cordinates in the S. purpuratus reference genome. 

5) protein_coding_genes.bed contains the coordinates of all protein-coding genes in the S. purpuratus reference genome. 

6) psg.csv contains a list of the positively selected genes identified by Kober and Pogson (2017).

7) tu_2012_go.csv contains the list of S. purpuratus GO terms created by Tu et al. (2012)

8) unique_exons.bed contains the coordinates of the unique exons in the S. purpuratus reference genome.
